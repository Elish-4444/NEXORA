# NEXORA Landing Page

A premium dark landing page for NEXORA, a digital solutions startup that helps startups and small businesses launch modern websites, improve their online presence, and automate workflows.

## Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript

## Features

- Sticky glassmorphism navbar with mobile menu
- Premium hero section with CTA buttons and dashboard-style visual
- Services section with reusable service cards
- Why Choose Us section with feature highlights and stats
- Testimonials section with client quotes
- Footer with contact details, quick links, and social placeholders
- Fully responsive layout for desktop, tablet, and mobile

## Setup

```bash
npm install
```

## Run Locally

```bash
npm run dev
```

## Production Build

```bash
npm run build
```

## Vercel Deployment

1. Push the project to a Git provider supported by Vercel.
2. Import the repository into Vercel.
3. Keep the default Vite settings.
4. Build command: `npm run build`
5. Output directory: `dist`
6. No environment variables are required.
7. Deploy.
